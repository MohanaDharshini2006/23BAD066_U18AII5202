#  Student Performance & Attendance Dashboard

An interactive dashboard designed to analyze **student attendance, internal assessments, assignments, and final grades** across departments, semesters, and subjects.  
This project helps educators and administrators quickly identify trends, performance gaps, and attendance patterns.

---

##  Project Overview

This dashboard provides a consolidated view of academic performance using key metrics such as:
- Attendance percentage
- Internal test scores
- Assignment marks
- Final grades

It enables **data-driven decision-making** by allowing users to filter and compare results across multiple academic dimensions.

---

##  Key Features

-  **Department-wise Performance Analysis**
-  **Subject-wise Attendance Comparison**
-  **Semester-based Filtering**
-  **Automated KPIs**
  - Average Internal Test 1 Score
  - Total Internal Test 2 Marks
  - Average Attendance Percentage
-  **Student-level Drill Down**
-  **Visual Insights using Charts & Tables**
